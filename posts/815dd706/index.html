ESP8266气象站食用指南
2021-01-05 00:00:00 # 教程 # ESP8266 # 物联网 # Arduino # 气象站

前言

我也不知道说些什么……

准备

硬件

开发板

开发板这里我们选择ESP8266，一般常见的有两个版本，CP2102和CH340，根据你个人实际情况选择即可。哪个便宜就买哪个。

屏幕

屏幕这边我们选择0.97英寸的OLED屏幕，0.97英寸的OLED屏幕应该是最便宜的屏幕了吧，虽然屏幕较小，但是用于气象站也是够了的。

你也可以根据的实际需要选择其他屏幕，但可能需要修改项目的代码，不太建议小白尝试！

传感器

传感器的选用的是DHT11温湿度传感器，记得选择带转接板的，这样会更方便些。

你也可以选择直接使用DHT22来提高精度，或者再买一个DS18B20来增加测温的精度（一般情况下，DHT11+DS18B20会比单独的DHT22更便宜）。

杜邦线

选择母对母的杜邦线，只需7根即可。但是一般选择40P的会更实惠，可直接拆成单独的杜邦线。

其他

  • 一台能上网的电脑。（当然USB口也要能正常使用的）。

  • MicroUSB数据线。用于烧写数据，充电线是不能用于烧录的。

软件

驱动

根据开发板的芯片选择对应的驱动程序，如CP2102和CH340有不同的驱动程序，自行百度或者询问客服获取。

Arduino IDE

用于编辑和烧录程序，在官网 下载适合你的版本即可

当然编辑和烧录你都可以选择其他的程序，因为Arduino IDE的速度是真的慢呐= =

api

我们选择的是和风天气 的api，在官网注册 并根据开发文档 创建你的项目获取Key。

然后在城市列表 里找到你所在的城市，复制相应城市的Location_ID。

你也可以选择其他api，但是需要改项目代码，并不建议小白尝试。

项目

你可以使用我提供的源码，不是我原创的！

来源：一个酷安老哥 发的一篇图文 里分享的代码。

这位老哥好像是用GitHub里的一个项目 魔改的。

当然，在GitHub或者其他地方寻找ESP8266气象站的项目，根据需要加以修改。

过程

安装驱动程序

这个就不多说了吧，如果不会自行百度！

然后电脑连接ESP8266，在设备管理器（Windows自带的那个，自行百度查找打开方法。其他系统自行摸索）查看端口号。

配置Arduino IDE

  1. 打开Arduino IDE

  2. 在左上角找到文件—首选项—附加开发板地址并输入这个网址 即可。

  3. 在左上角找到工具—开发板管理器搜索esp8266并安装。

  4. 在左上角找到项目—加载库—添加.zip库...找到文件里的libraries文件夹并加载。

修改配置文件

在文件里的Weather文件夹内打开Weather.ino文件。

在左上角找到文件—首选项选择显示行号。

  1. 找到78行的const char* HEFENG_KEY="";，在""中输入刚刚在和风天气中获取的Key。

  2. 找到79行的const char* HEFENG_LOCATION="";，在""中输入刚刚在和风天气中找到的Location_ID。

烧录程序

  1. 在文件里的Weather文件夹内打开Weather.ino文件。

  2. 在左上角找到工具—开发板将开发板改为ESP8266 Boards里的NodeMCU1.0（ESP-12E Module）。

  3. 将ESP8266连接至电脑。

  4. 在左上角找到工具—端口改为刚刚在设备管理器中找到的设备端口号。

  5. 在左上角找到项目—上传。

  6. 一个船新のWeatherStation就诞生了！

硬件接线

屏幕接线

  • D3—SDA

  • D4—SCL

  • 3V3—VCC

  • GND—GND

传感器接线

  • D6—DAT（这个可以在Weather.ino文件中修改）

  • GND—GND

  • 3V3—VCC

通电，然后按ESP8266上的RST。

WiFi配网

  1. 连接ESP8266的WiFi（默认为WeatherStation，你可以在Weather.ino文件中修改）。

  2. 在浏览器中输入WiFi的IP（如果不知道WiFi的IP怎么查看，自行百度）。
