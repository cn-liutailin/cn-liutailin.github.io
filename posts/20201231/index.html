序
#

2020年是极不平凡的一年，这一年发生了太多的事。时间过得飞快，转眼间，2021年已经到来。2020年给我们带来了太多的感动和遗憾，有太多的事情值得我们去纪念和思考。但是，2020年过去后，真正能够留下的又有些什么呢？也许，只剩下这篇回忆录了吧。

谨以此文纪念那不平凡的二零二零年。

不一样的年
#

由于那段时间基本上与网络世界隔绝，所以还不知道疫情的到来。

偶尔几次出去，也只是觉得街道上跟往年相比，有些冷清。直到春节过后，才发现疫情的严峻形势。

最长寒假
#

由于疫情的影响，假期一次又一次的延长。不过我习惯于宅在家中，所以基本上没对我造成太大的影响。

时间一天一天的过去，我也逐渐觉得家里闷得慌。有时甚至想念学校、想念同学了。

网课时代
#

网课时光可能是我初中的时候最疯狂、最快乐的一段时光吧。由于疫情原因不能返校，所以学校开始上网课。网课这玩意儿我还是不陌生的，毕竟之前也购买过一些网课。

看到评论区有熟悉的同学时，也会感到很开心。起初我还是有在认真的上课的，之后就一边开着网课，一边打开GBA模拟器来玩，玩《火焰纹章》、《逆转裁判》之类的。

之后我们上的那个网课机构的政策变了，开始收费，于是我们学校就索性让自己的老师开网课。新的网课平台比较恶心，会锁屏，于是直接按Ctrl+Win+D切换桌面。

之后朋友写了一个解开锁屏的软件，就开始用他的软件。他还在公开版中装了后门，每隔一段时间截图发送至他的服务器。其实大家基本上都没在认真上课，用这个软件的人谁会认真上课啊？ 干各种事情的都有。甚至还发现了一个人在玩 galgame 。

后来索性就不打开网课软件。从根本上解决了问题 。由于碰巧遇上之前玩过的mc服务器重新开始小范围公测，于是就开始玩起mc来。服务器的反作弊基本上和没有一样，于是就在服务器里做神仙 。每天挖矿、打怪、钓鱼……十分地悠闲自在。

不过后来被老师发现没上课，就还是把网课挂在了后台。

等到快开学的时候，我又打开网课软件去看。由于这个网课平台不成熟，配置文件什么都没有加密。但是我一直在玩mc，所以就没去研究这个。一些人就开始研究这个配置文件，一些研究成果也会相互分享。于是我一打开就看到一堆改昵称的（原本是实名的）。还有刷屏的（评论区讲课时间老师一般都是关闭的）。

看来网课不仅让他们学到了课本上的知识， 或许没有 ，还让他们学到了一些在课本上学不到的知识。

最后的初中时光
#

重返校园
#

疫情开始好转，各地开始复工复产，各类大中小学开始上课。我们也开始返校了。

还是那个熟悉的学校，还是那些熟悉的老师和同学，一个超长的寒假过去后，竟然这么的想念他们。

回到班上，班主任开了一个 “励志的班会”。之后便是和以前一样，开始了一天又一天的课程。由于疫情原因，学校发放口罩，测体温，并且取消了晚自习，好耶！

取消体考？！
#

我们为了应对体考，每天大课间学校都会组织跑步、跳远、跳绳等运动。但是我并不擅长体育，每天这个时候就会感到非常地痛苦。

所以在大课间的时候我 经常摸鱼，有时候甚至直接不去上大课间。

当我们得知取消体考的信息时，我是挺高兴的，这对不擅长体育的我来说是100%有利的。

中考延期？！
#

由各种原因，我们省中考延期了，延期一个月。之前老师还和我们说大概率不会延期，所以这是遇到了小概率事件？

我倒是挺不希望延期的，眼看着就快要中考了，结果又延期一个月。剩下这一个月根本就没有什么复习的欲望，而且暑假也就相应的被压缩了一个月，计划好的暑假生活又需要更改了。

理化实验操作——从理论到实践
#

中考前大概一个月我们举行了理化试验操作考试，这个考试基本上人人都能拿满分。

公布考试内容后，我们就开始针对考点，学习理化实操。每天都会播放实验视频，上物理、化学课之前也会放实验视频。

考试之前大概做了两次实验，都不太记得做了什么实验了。印象比较深的就是，我们做一个物理实验需要带打火机和棉签，班里一些猛士直接点燃棉签，猛吸一口，然后就呛的直咳嗽，我得出结论：抽烟有害健康 。

等考试来临的时候，我还是有点紧张的，因为看视频、做实验的时候都不太认真其实 。开始是化学实验，抽到了一个比较简单的实验。不过还是出了点小意外，写结论的时候过于紧张，化学方程式忘记配平了，而且一直想不起来需要配平，直到老师提醒我才写出来了。出来我才想起，我好像没洗试管，不过最终还是满分。物理实验抽到一个相较于其他的物理实验稍微难一些的，不过有了化学实验的教训，没有那么紧张了，一切进展顺利。

最后一次考试
#

学校组织了最后一次考试，考了物理和化学。

但是当我回到家时，惊讶的发现我创建的学校年级群里，考试之前就有人发了化学答案。

这立马引起了我的警觉，我把截图发给化学老师。然后撤回群内相关信息，并且将发答案的人踢出群内，打开全体禁言，然后跟群友们说明群内禁止发答案。一气呵成。

不过还是没能阻止事情扩散下去，这之后的很多人涌入我的群。虽然我的群开了禁言，不过还是没能阻止他们通过其他途径传答案，于是，越来越多的人获取到了答案。

不过学校老师们应该注意到这些事情了，就没有将这次考试的成绩公布。我听说不会公布成绩，于是我就在语文作文中写了这件事。

考完试后，我也试着去查泄题者的账号，不过显然那只是一个小号，就不了了之了。晚上，这件事逐渐平静下来，我就把全体禁言关闭了。

最后的复习时光
#

已是七月份了，平时就早已放暑假了。初一、初二也早已放假，整个学校就剩下我们初三在这酷暑之下上课。

时间一天一天的过去，每天重复着那几门科目，有些科目我是会认真听一听的。但是有些科目真的学不下去了，我就躲在书本后面、趴在桌子上发呆，发着发着呆，就睡着了。因为都快考试了，老师也基本上是睁一只眼闭一只眼的。有时候一觉起来，就下课了，然后起来上个厕所、洗把脸，等到上课，又趴在桌子上进入幻想乡了。

考试前一天，我们搬到了休息室。休息室，可不是什么休息的好去处，由于快要考试，楼下做了考场，留着最顶层做休息室。顶层，就意味着最热。最后一天了，我也不知道自己从哪里迸发出来的动力，竟然开始复习了。可是又有些迷茫，一翻开书，自己不知道的东西太多太多了，一时竟不知从哪里看起。

班主任允许大家下午不来复习，但是需要让家长请假。我是不想回家复习的，而且我家长肯定不会给我请假的。到了下午，来了的同学其实挺多的，班主任给我们来了的人每人买了一瓶饮料，￥3的饮料，只要学校小卖部有，可以任选。也是挺惊喜的。

考试&假期
#

中考
#

中考那天，阳光明媚。

第一场考试考的是语文，在初中，语文作文我是我最害怕，但是那天的作文并不难，而且我也直接套用了之前背的作文，虽然我知道套作得不了多少分 ,但是我初中基本上不会写作文，初中作文是记叙文，我编不出好的故事来。

考试结束后，并没能直接放学回家，学生们聚集在校门口（好像是因为卷子没有整理好不能开门）。

学校开门后，因为比平时放学早，所以我和朋友在桥底聊了一会儿天（因为桥底比较凉快，风挺大，而且挡太阳）。具体聊了什么我也忘记了，大概是关于暑假和以后的规划吧。第二天我们也在桥底下聊天，遇到了一个巡逻的老师，那老师认出我们来了（我们前几周帮过这个老师做事，每个班要给班班通装一个监测软件），老师催促着我们离开，因为河边可能比较危险，怕学生下水游泳。

中午一吃过饭就去了学校，到了学校就去小卖部。买了曼妥思和饮料，曼妥思我的印象比较深，里面有黄色和绿色的糖，黄色的是柠檬味的，绿色的是莫吉托味的。那个时候我还没听过周董的《Mojito》，只是觉的这糖的味道挺有趣的。初嚼有薄荷的苦涩，之后有一丝淡淡的糖的甜意。如同我的初中生活一般，苦涩中夹杂着甜蜜。之后，我将糖分给了一些朋友，和一起他们分享这苦涩的甜蜜。

下午考的是英语，初中最不懂的应该就是英语了。英语在初一的时候还好，不过从初二开始英语成绩就开始下滑。不过也是尽力去考了，虽然词汇量达到了瓶颈，不过文章还是能大概的读懂的。

结束了？！
#

最后一天考的都是我比较擅长的科目，整场考试下来十分的顺利。

考完后回到我们班的休息室，举行毕业典礼。发毕业照，每个人都发了一些零食和留念的本子。之后班主任给每个人发毕业证，我感觉这一切都太突然了，这段时间就好像加速了似的。班主任说了些话后，就告诉我们可以回去了。

待了三年的母校，现在就要分别，我有些不舍，有些不甘，但也无可奈何。

离开了学校，我在校门口的小卖部买了瓶汽水，这可能是我最后关顾这家店了。我和经常一起回家的朋友们同行，一个朋友说今天的考试对我来说很不利，因为题目都很简单。我好似被泼了一盆冷水，把刚才考的基本上都会的兴奋给浇灭了。我倒吸了一口凉气，仔细一想，确实如此，这次的中考我是十分失败的。这也并不是因为发挥失常，究其原因其实就是懒惰，没有用功读书。

《我的世界》服务器建设与管理
#

朋友拿自家电脑开了个mc服务器，我们很早之前就已经计划好了。邀请了许多朋友来玩，不过也没来太多。整个服务器就我和他在管理，他负责整个服务器的管理，我是负责游戏内的管理。例如：建造服务器的主城、给服务器商店定价……

进服务器的人我们大多都认识，所以管理起来比较的容易。但是商店定价有些不合理，有肝帝通过挖煤成为了服务器首富。我开始是想改服务器商店的定价的。但是，那时候我被叫去珠海玩。回来之后服务器大变样了，各式各样的建筑都建起来了。因为服务器是假期服，开学就会关闭，所以就没打算让他们太肝，商店的定价不合理的问题也就忽略掉了。

那个时候，玩家形成了集聚，主要集中在“祖安镇”。离主城很近，主城又是另一个建筑集聚。我还找了很多刷怪笼做成刷怪塔，搞了个全物品分类机，打算建一个工厂。但是后来由于太麻烦了，就放弃了。看到了TIS服务器建的赛船场（滑冰场），我就也在服务器里建了一个赛船场。

珠海之行
#

珠海，我小学在那里读书。

我很多个暑假都会去珠海，还是很怀念那个城市的。暑假在珠海待了大概两三个星期。

珠海有名的地点几乎都去过了：港珠澳大桥、日月贝、珠海渔女……

这次在珠海看到了很美的晚霞，在公园遇到了黑和白天鹅、遇到了可爱的鸭鸭。早晨去受佛教影响的白莲洞公园感受别样情调，也会在夜里骑车去看海感受清凉海风，珠海是一个浪漫的城市。

成绩日
#

中考成绩出来了，我去查分。发现比平时考试的分高，稍微庆幸了一会儿。不过我知道，考的不会太好。

几天之后，各个高中公布学校的录取分数线。我看了一下，我的成绩应该是考不到城区内的高中，就准备走艺考上高中。

艺考之路
#

美术，几乎是从我记事起就开始学的一门学科。

但是，城区内只有一所高中招收美术生。开始我还兴致冲冲地去一中报考，结果发现这所学校根本不招收美术生。

于是，我就去二中报名。考题也很简单，是简单石膏几何体。中途还遇到了一些初中的同学。

成绩出来后，我以三十多名的成绩（文化+特长的综合分）考进了这所学校。这意味着我已经走上了艺考之路了，也意味着我要牺牲假期的时间去学校学美术。

我告诉了我的朋友们，朋友们都表示了祝贺。我问了问我去考试的时候遇到的同学，他是我初三的一个比较要好的同桌。他没有考上，就去四中继续艺考了。之后他好像是考上了四中，但我觉得挺遗憾的，他没有和我在一个学校。祝愿他能够考上一个令自己满意的大学。

感谢从小学到大的美术，给险些考不上高中的我，提供了一条新的道路。

最短暑假
#

由于延迟了中考，所以暑假时间也相应的被压缩了。暑假很短暂，但是我过的很丰富。

新的开始
#

从零开始高中生活
#

高中，一切都会重新开始。准备好大干一场了。

失望与后悔
#

到了二中发现，他并没有那么美好。它收最多的艺术生，但它并不是什么艺术的殿堂。几乎所有同学都是因为没高中而读才会选择来学美术，有些人甚至根本不了解美术。而我们高一整个学期也没有上过美术专业课。

还能更糟？！
#

我以为这一切已经够糟糕的了。结果我们分班之后，整个班被分去了地下室。那里没有一体机，走廊边上是肮脏的下水道，班主任由一个和蔼可亲的中年大叔换成了一个凶巴巴的年轻老师。

班主任是一位历史老师，还是在我们学校实习的实习生。我们叫他老林，老林开始时说话十分严肃，但是后来相处久了，我们也就逐渐的了解他了，老林其实是位极其负责任的老师，我们也就没有那么讨厌这位老师了。

军训
#

军训，是每个高中开学的必修课。因为疫情，我们的军训举行的比较晚，天气也不会太热。

军训套装穿着非常不舒服，尤其是鞋子，穿起来和铁板似的。军训的教官是个有趣但是有点凶还有点微胖的男人，军训时老林经常来看我们，还叫教官给我们加练。军训总是期待着下雨，我们甚至还画了求雨符，不过没什么作用就是了 ，祈求着雨的来临。前几天太阳都不是特别烈，有天下午，太阳异常地烈。老林在我们休息的时候给我们每个人买了矿泉水。

军训最后一天是军训成果展示会，不过我们班也没有获得什么荣誉。会后，教官们就匆忙地走了。第二天，虽然手脚都因为军训的缘故有些酸痛，但是挺多人想念教官的（包括我）。

加入学生会
#

我们学校有两大组织，学生会和志愿者。

我看了学生会的部门介绍，发现有一个部门——编辑部，非常适合我。因为我初中帮老师整理电子文档，office的操作我完全没问题，而且我对计算机很感兴趣。

之后我就报名了，老林是鼓励我们去参加的。不过他还是让大多数人去志愿者了，他对志愿者貌似没什么好感（甚至有一次志愿者开大会的时候把我们班的同学全部抓回来了）。

报名之后就是面试，需要自我介绍和回答问题。我自我介绍说的一塌糊涂，我都不知道我在说些什么，提问时我回答的倒还好。我们班另一个想进编辑部的同学没能通过。第二天又测试了打字，我对打字是很有信心的。最后我顺利的进入了学生会。

十字路口
#

在我们学校，考试约等于放假。我们只需考试时到考场就行，其他时间可以回家。

第一天我们考的是语文，语文什么的根本无需复习。但是早读我还是装模做样的在读书，读的是诗词。自从上了高中，不知怎么的，就对诗词很感兴趣。对我来说，读诗是一种快乐。

最后一天考的是英语，英语什么的，对我来说挺难的。有些迷茫，不知道自己在写些什么。考完试后，由于距离晚自习还有挺长一段时间的，吃过饭后，同学提议去散步，我没什么事，就和他一起去散步了。

到了三中附近的一个十字路口，遇到了初中的一个同学。我沿着街径直走了过去，听到好像有人在叫我。回过头来，发现原来是我初三的同学，笑容可掬的看着我，我自从上高中以后就没有再见过其他学校的初中同学了，还真是十分想念他们 。她的笑容如同一缕春风，吹散我心中的阴霾。我忽然想起我的初中生活，心中有数不尽的话想说，可是话到嘴边，又不知道该从何说起。愣了几秒，就转身离去了。

回到学校后，我又想起了她。想到了她的眼睛，她的眼中仿佛若有光。而我在这之前，就如同迷失在深夜的十字路口，非常的迷茫，不知道自己该走向何方。而她就好像一盏路灯，让我找到了前进的方向。

运动会
#

一年一度的运动会又开始了，但我并不擅长体育。通常来说，我是这类活动的绝缘体。但这次不一样，是作为学校的工作人员参与的。

运动会前夕，学生会开了动员大会。每个部门都分配有相应的工作，编辑部被分配的是：制作校报、失物招领和机动组。

第一天集合，很尴尬的迟到了（我以为是开幕式后集合，结果在这之前就已经集合了）。第一天被分配学习做校报，很容易的就领会到其中的精髓。第二天被分配去失物招领处，遇到了很多有趣的人和有趣的事。第三天被分配去了机动组，其实就主要就是站岗，摸鱼ing 。

运动会这三天对我来说有点劳累，以至于第一天晚上一回家就睡着了，凌晨两点醒来洗了个澡就又回去睡觉了。虽然有些劳累，但是过的还算充实，倘若是在初中，我应该三天都在学校里游荡，无事可做。运动会时做的校报是我们部里面做的最出色的。于是乎，我就成为了我们部门的负责人。

朗诵比赛（不尴尬挑战）
#

语文老师在上课时提到了学校搞了一个朗诵比赛。班长统计了参赛的人数，我原本没想参赛的，但是后来想起了海子的《面朝大海，春暖花开》，又萌发了想去尝试的念头。结果就稀里糊涂的报了名，我就不应该报名的 。

到了晚自习，因为只能有一个人去代表全班去参加单人朗诵，老林让我们在全班面前比一比。我原本读的已经很烂了，但是没想到另一个读的更烂。结果，我就成为代表我们班的参加比赛的那个了。

但是听语文老师说过，原创的会更好一点。于是我就在晚自习写了首诗，写完后就拿给老林看。

老林说他觉得写的还行，叫我拿去给语文老师看。我在语文课后给老师看，老师先鼓励我说还行，并指出了几点不足之处。我问她我是拿我写的诗去参赛还是拿海子的诗去参赛，老师可能是想让我自己选择，就说海子的“也很好”，含糊过去了。之后我的诗给了我们班的同学看，他们说写的挺不错的。又拿给初中的好朋友们看，他们也认为写的还行。拿给父亲看，他也很鼓励我去参加这种活动。于是乎，我那时候就十分愚蠢的认为自己是“带文豪”。

我自认为我写的这首诗还不错，但是会写诗不一定会念诗。我并不认为我的朗诵、演讲的能力强，我只是一个写诗的人，我不想也没有这个能力，将我的文字赋予情感后表演给别人，我只是希望他们读到某处而产生情感共鸣，仅依靠文字打动人心，仅此而已。海子的很多诗，我读了后，内心深受触动。而我又去听别人朗诵他的诗时，我又没有了那种共鸣，因为他的读法和我的理解是有些偏差的。也许，这就是文字的魅力。

朗诵比赛和我预想的一样糟糕，甚至还要更糟。整个朗诵比赛中，就只有我一个人用原创诗参赛。而且，他们选用的题材大多都是爱国类还有古诗词之类的，显然我的诗不符合老师所需要的题材。我是很尽力的带入情感去朗诵了，但是，很自然的，我落选了，我也没有太过于难过，毕竟我知道自己并不适合朗诵。

这是一次非常 社死 难忘的经历，不过我以后肯定不会去参加朗诵类的节目了。这个故事告诉我们：会写诗不一定会念诗。

老林的离去（泪目
#

由于老林是实习生，与我们相处两个多月后就要返回他的大学了。

老林虽然开始时很严肃，当时之后的课上他还是很幽默的。也是一个极为认真负责任的班主任。回想起老林的一些趣事仍然历历在目：有一次老林给我钱叫我帮他买饮料，我和几个同学就一起出去了，就买了百事可乐这是学校门口能买到最难喝的饮料之一，然后我们几个同学轮流摇可乐，不过老林还是很有经验的，可乐并没有喷溅出来。

最开始我们班的同学还挺害怕甚至厌恶他，我甚至在见面第一天就骂过他还给他听见了 。后来换了班主任，最后还是发现老林最负责。老林每天很早就在教室里等着大家来午休、教给大家其他老师不会叫的礼貌（就是进门看到老林在班上要从正门进去并且要打报告）、讲一些充满哲学（不是那个哲♂学）意味的大道理……

老林最后一个晚自习，大家纷纷送老林礼物留念。我送了一幅画，画的是五彩斑斓的雪山和森林，因为老林在东北上大学，就祝他以后的时候能像这画中的一般，五彩斑斓。老林最后对我们留了一句赠言：未来可期。上一句记不得了 。

表白墙的建立与倒塌
#

一次在初中朋友们的群中，有人发了一中的表白墙网站。是我的朋友搭建的墙，就是暑假mc服的服主 。我打开来看，做的还可以，用户也比较多。

晚上我辗转反侧，想到我们二中也可以整一个啊。第二天我就立马开整，由于当时手头上没有服务器，我就直接用的静态网页，再放出投稿按钮，让他们投稿。投稿其实就是利用的问卷星，做了一个问卷，然后把他们发的内容复制下来。然后直接 git push 推送到 GitHub Page，这样效率非常低，但在当时没有服务器的情况下是最优解了吧。宣传方面我找了二中的情报网（挺中二的名字的），因为他在我们班，所以就让他宣传了一下。

很快就有了少量用户，当时他们大多只是抱着好奇心来访问这个网站的。很快就出现了一些问题，由于服务器在美国，所以从中国访问很慢，加之在网页中塞入过多的 JavaScript 和 CSS ，导致他们访问缓慢。

过了大概有一个月，几乎没有人访问网站了，我索性就把它给关闭了。

后记 ：给2020画上句号
#

2020年对我来说是光怪陆离的一年，对我来说绝不只有疫情这件大事。这一年的大部分时间我都是在迷雾之中度过的，我的世界里没有光，不知道该走向何方，整日都在虚度光阴，没有取得什么成绩。但是，这一年我也收获了许多。这一年较往年相比，应该是我成长最快的一年，不单是身体上的，更重要的是心理上的、精神上的成长。

虽然2020年给我带来了很多遗憾，以至于我至今仍偶尔想起2020年的某些片段，后悔的想：如果我那时候那么做了会怎样啊？

但是如果我再次穿越回到2020年我还会愿意吗？我想是的，虽然是糟糕的一年，但我仍觉得值得停下来细细思考。

但是生活，没有如果。让我们给2020年画上最后的句号，就让他尘封在这文字之中吧。
